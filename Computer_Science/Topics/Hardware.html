CPU (Central Processing Unit)

The CPU is the brain of the computer it runs every instruction of the computer.

The CPU is used in all kinds of things. E.G.

TV - uses it to decode the video and to run the OSD (On Screen Display).
Washing machine - to control the heat and to keep the machine in time.

This uses the Microprocceser which unlike old computers where the CPU could be the size of a room a microprocceser can be the size of a few centimeter.
