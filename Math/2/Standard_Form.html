Standard Form

When we use standaed form we often use it to express a very large number in a number that is < 10 but > 1.

So we would espress a standard form number with ×10ᵃⁿʸ ᴺᶸᵐᵇᵉʳ So for example 5.8765×10⁵ which would be 587650 so we can convert how many numbers in the power to the amount of places to move the decimal place if we have the decimal place go futher than the number we just add zero(0).
