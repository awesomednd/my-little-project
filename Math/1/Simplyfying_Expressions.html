Simplyfying Expressions

The parts of an algebraic expression are called terms. Terms are separated from each other by + or - sing. So a+b is an expression with two terms, but ab is an expression with only one term.

Adding and subtracting like terms

Terms with exactly the same variables are called like terms. 2a and 4a are like terms; 3xy² and -xy² are like terms.

So we would group them into like terms e.g. 4a + 2a + 3a so ther is 4+2+3 = 9 then we add the a so 9a so now time for a seccond example 5q - 4q + 8p + 7q - 2p so lets group the q so 5 - 4 + 7 = 8q and then the p 8 - 2 = 6p so it would be 8q + 6p
