Using Letters to Represent Unknown Values

In primary school you used empty shapes to Represent unknown Numbers.

In algebra, you use letter to represent unknown numbers. So you could write the number sentences above as: 2+𝑥=8 and a+b=10. Number sentences are called equations. You can solve an equation by finding the values that make the equation true.

When you worked with the are of rectangles and triangles in the past, you used algebra to make a general rule, or formula, for working out the area, A:

Area of a rectangle = length X breadth, so A = LB

Area of a triangle ¹/₂ base x height, so A=¹/₂BH or A=BH/2

Notice that when two letters are multiplied together, we write them next to each other e.g. LB, rather than LxB
